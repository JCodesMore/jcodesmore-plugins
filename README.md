# JCodesMore Plugins

Claude Code plugin marketplace by [JCodesMore](https://github.com/JCodesMore).

## Install

```bash
/plugin marketplace add JCodesMore/jcodesmore-plugins
```

Then install any plugin:

```bash
/plugin install youtube@jcodesmore-plugins
```

## Available Plugins

| Plugin | Description | Repo |
|--------|-------------|------|
| `youtube` | YouTube tools — search, transcripts, video info, channel browsing, playlists | [youtube-mcp](https://github.com/JCodesMore/youtube-mcp) |
